Customer Analytics Dashboard Project

Project Overview

This project focuses on analyzing customer purchasing behavior, membership trends, customer satisfaction, and revenue insights using a complete data analytics workflow. The project was developed by combining Python for data cleaning, MySQL for data analysis and query solving, and Power BI for interactive dashboard creation and business presentation.

The goal of this project is to transform raw customer data into meaningful business insights through proper data processing, querying, and visualization techniques.

Project Workflow

1. Data Cleaning using Python

The raw customer dataset was cleaned and transformed using Python in Jupyter Notebook.

Tasks Performed:
- Removed null values
- Checked duplicate records
- Corrected data types
- Created age groups
- Standardized categorical values
- Prepared dataset for database integration

2. Data Storage & Query Analysis using MySQL

The cleaned dataset was imported into MySQL for structured analysis.

SQL Operations Performed:
- Customer segmentation analysis
- Membership-based customer count
- Revenue analysis
- Discount usage analysis
- City-wise customer analysis
- Top spending customer identification
- Customer satisfaction filtering
- Aggregate functions using COUNT(), AVG(), SUM(), GROUP BY, ORDER BY, LIMIT, and WHERE

Power BI Dashboard Development

An interactive Customer Analytics Dashboard was developed in Power BI to visualize business insights effectively.

Dashboard Features:
- KPI Cards
- Revenue Analysis
- Membership Distribution
- Gender-wise Customer Analysis
- City-wise Revenue Comparison
- Customer Satisfaction Metrics
- Discount Usage Insights
- Interactive Filters & Slicers
- Professional Dashboard Layout

Key Business Insights

- Gold membership customers contributed higher revenue
- Certain cities showed stronger customer engagement
- Discounts influenced customer purchasing behavior
- Customer satisfaction varied across demographics
- Spending patterns differed by age group and membership type
